Visão Geral

O Organizador de Arquivos POSTECH é uma ferramenta desktop para organização de arquivos de vídeo acadêmicos. Desenvolvida com Electron + Rust, oferece automação completa para padronizar estruturas de pastas, gerar relatórios, emails e planilhas.

Versão

A versão atual da aplicação é exibida na sidebar e nos Créditos. O número de versão é centralizado em uma variável para facilitar atualizações futuras.

💾 Backup / Restauração

Crie um backup completo de todas as suas configurações, atalhos, perfis de cores e projetos salvos.
